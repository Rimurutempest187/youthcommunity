# Church Community Bot

File-based Telegram bot for church community features: quizzes, verses, events, contacts, prayers, reports.

## Quick start

1. Copy `.env.example` to `.env` and fill `BOT_TOKEN`.
2. Create a Python virtualenv and install dependencies:
   ```bash
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
